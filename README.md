Recipe Maker (Spring Boot Web Application) :
Recipe Maker is a full-stack web application that demonstrates the following:
 - Uses Spring AI to interact with AI models via OpenRouter to generate recipes at users request
 - Handles file upload using MultipartFile allow users to upload/change their profile pictures
 - Utilizes OpenPDF to allow users to export/download their recipes in PDF form to share
 - Uses JDBC and JPA models to store data on databases, allowing the data to be saved for future use

The project comes with 2 user instances preinstalled with the ability to register more users.
- root@example.com
- guest@example.com
Both with password 1234

Technologies Used:
- Java
- Bootstrap
- Spring Boot
- Spring AI
- OpenAI model
- JPA + JDBC
- OpenPDF
